# 计量单位
## 长度
| 类型 | 单位 | 说明 |
| :-: | - | - |
| 纳米 | nm | 1/1000微米，10^-9米，相当于4倍原子大小 |
| 微米 | μm | 1/1000毫米 |
| 毫米 | mm | 1/1000米 |

## 时间
| 类型 | 单位 | 说明 |
| :-: | - | - |
| 纳秒 | ns | 1/1000微秒，10^-9秒 |
| 微秒 | μs | 1/1000毫秒 |
| 毫秒 | ms | 1/1000秒 |
